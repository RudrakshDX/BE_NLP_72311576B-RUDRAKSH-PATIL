1. Problem Statement

Perform tokenization using the NLTK library:

Whitespace Tokenization
Punctuation-based Tokenization
Treebank Tokenization
Tweet Tokenization
MWE Tokenization

Also perform:

Stemming using Porter and Snowball Stemmer
Lemmatization using any technique
Input / Dataset

A sample sentence is used as input.

Example:

NLTK is a powerful library! It helps in tokenization, stemming, and lemmatization.

2. Explanation
Tokenization

Tokenization splits text into smaller units called tokens using different methods such as whitespace, punctuation rules, linguistic rules, and tweet-specific patterns.

Stemming

Stemming reduces words to their root form.

Porter Stemmer
Snowball Stemmer
Lemmatization

Lemmatization converts words into their base (dictionary) form based on context.

3. Conclusion

This assignment demonstrates basic text preprocessing techniques using NLTK. These methods are essential for Natural Language Processing tasks.
