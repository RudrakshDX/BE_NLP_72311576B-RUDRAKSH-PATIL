# NLP Practical Assignment 3

## Problem Statement
Perform the following preprocessing steps on the dataset:
- Text Cleaning  
- Lemmatization  
- Stop Words Removal  
- Label Encoding  
- TF-IDF Representation  

Save the processed outputs.

---

## Dataset
Dataset used: News Dataset  
Source: https://github.com/PICT-NLP/BE-NLP-Elective  

---

## Explanation

### Text Cleaning
Cleaning involves removing unwanted characters such as punctuation, special symbols, numbers, and converting text to lowercase.

### Lemmatization
Lemmatization converts words to their base or dictionary form using linguistic rules.

### Stop Words Removal
Common words like "the", "is", "and" are removed as they do not add significant meaning.

### Label Encoding
Categorical labels are converted into numerical form so that they can be used in machine learning models.

### TF-IDF
TF-IDF (Term Frequency–Inverse Document Frequency) converts text data into numerical vectors based on word importance.

## Implementation Overview
1. Load dataset

4. Remove stop words
